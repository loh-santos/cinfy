# CatálogoFlix

## Objetivo
Site de catálogo de filmes desenvolvido como atividade avaliativa da disciplina de Desenvolvimento Web, utilizando apenas HTML, CSS e Javascript puro (sem frameworks ou bibliotecas).

## Tecnologias utilizadas
- HTML5 semântico
- CSS3 (Flexbox e Grid)
- Javascript puro (DOM, eventos e validação de formulários)

## Estrutura das páginas
- **index.html** - Página inicial (Home), com apresentação do site.
- **sobre.html** - Página Sobre, com informações do projeto e integrantes.
- **filmes.html** - Catálogo de filmes, com busca por título, filtro por gênero e modal de detalhes.
- **contato.html** - Formulário de contato com validação de campos e mensagem de sucesso.

## Funcionalidades implementadas
- Catálogo de filmes montado dinamicamente via Javascript (DOM).
- Busca de filmes por título em tempo real.
- Filtro de filmes por gênero.
- Modal (janela) com detalhes do filme ao clicar em um card.
- Formulário de contato com validação dos campos (nome, e-mail, assunto e mensagem).
- Mensagem de sucesso exibida após o envio do formulário, simulando o envio dos dados para um banco de dados (armazenamento no localStorage).
