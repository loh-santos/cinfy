Sobre o projeto

O Cinefy é um site de catálogo de filmes desenvolvido como atividade avaliativa da disciplina de Desenvolvimento Web.

O objetivo do projeto é praticar os conceitos fundamentais de HTML, estilização com CSS e Javascript puro, sem o uso de frameworks ou bibliotecas externas.

O site permite que o usuário navegue por um catálogo de filmes, pesquise por título, filtre por gênero e visualize detalhes de cada filme em uma janela de detalhes (modal). Também conta com uma página de contato com validação de formulário.

Tecnologias utilizadas

  • HTML
  • CSS
  • Javascript puro
